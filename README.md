# OpenCV Programs

A collection of beginner-friendly OpenCV programs written in Python to practice fundamental image processing and computer vision concepts.

## Features

# Program 1 - Image Processing

* Convert an image to Grayscale
* Apply Gaussian Blur
* Perform Canny Edge Detection
* Dilate image edges
* Erode image edges

# Program 2 - Image Resizing & Cropping

* Read an image
* Resize using custom dimensions
* Resize using scale factors
* Crop a selected region of the image

# Program 3 - Drawing Shapes

* Create a blank image
* Draw filled circles
* Draw rectangles
* Draw lines
* Add custom text to an image

# Technologies Used

* Python 3
* OpenCV (`cv2`)
* NumPy


# Learning Outcomes

Through these programs, I learned:

* Basic image processing techniques
* Image resizing and cropping
* Edge detection using the Canny algorithm
* Morphological operations (Dilation & Erosion)
* Drawing geometric shapes
* Adding text to images
* Working with OpenCV and NumPy



These programs were created while learning OpenCV fundamentals and computer vision concepts using Python.
